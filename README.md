# Google Workspace Hackathon 2024 - Group Tom & Jerries

This repository stores all the code and solutions for Group Tom & Jerries' participation in the Google Workspace Hackathon 2024. The solutions focus on the marketing department, automating and scheduling social media posts using Google Calendar, Google Sheets, Gmail, and App Script.
